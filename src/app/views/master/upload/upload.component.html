NOTES:

1. Position Organisasi dan Pegawai : digunakan untuk create data posisi SIMKP yang berasal dari data SAP.
Cara meng-uploadnya dengan melampirkan 2 template yaitu template organisasi dan templat pegawai. Data organisasi yang diupload akan menjadi data organisasi yang terbaru, jika ada posisi dan jabatan yang lama tidak terdata di template maka enddate posisi dan jabatan lama akan otomatis berakhir sesuai periode upload.

2. Saat upload pastikan kolom sesuai dengan template yang disediakan system ini dan Nama Sheet harus Sheet1 dan kode SAP di kolom J.

3. File hasil Download-an dari SAP harap dicopy - paste ke excel baru setelah itu Save As 2003.

4. Tipe Data Startdate dan Enddate di template data pegawai harus bertipe Date.
